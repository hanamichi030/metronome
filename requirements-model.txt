# --- Bittensor: MUST be the 10.x line (changed 2026-07-17) ---
# The finney runtime upgraded netuid to a Composite/newtype (NetUid{u16}).
# bittensor 9.x's substrate client (async-substrate-interface 1.x + bt-decode
# 0.8.0, the newest that exist on that line) CANNOT encode it and dies with
#   "ValueError: Invalid type for data: 126 ... Composite({u16})"
# on metagraph.sync()/serve_axon. Only async-substrate-interface 2.x (scalecodec
# encoder) handles it, and that requires bittensor 10.x. Verified live on
# bittensor 10.5.0 + ASI 2.2.1.
#
# bittensor 10.x skips CLI parsing unless BT_NO_PARSE_CLI_ARGS=false is set —
# see miner.env.example. Without it, config.neuron is None -> startup
# "AttributeError: 'NoneType' object has no attribute 'name'".
bittensor==10.5.0
async-substrate-interface>=2.0.4,<3.0.0
bittensor-drand>=2.0.0,<3.0.0
bittensor-wallet>=4.1.0
